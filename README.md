# GitHub Profile Viewer

Приложение для просмотра профиля GitHub пользователя с отображением статистики и репозиториев.

## 🚀 Технологии

- React 18
- TypeScript
- CSS3
- GitHub API (демо-режим)

## ✨ Функции

- 🔍 Поиск по GitHub username
- 👤 Отображение информации о профиле
- 📊 Статистика (репозитории, подписчики, подписки)
- 📦 Список популярных репозиториев
- 🎨 Адаптивный дизайн
- 🔗 Прямые ссылки на GitHub

## 🛠️ Установка

```bash
npm install
```

## 🏃 Запуск

```bash
npm run dev
```

## 📦 Сборка

```bash
npm run build
```

## 📝 Примечание

Приложение работает в демо-режиме. Для получения реальных данных добавьте API ключ GitHub.
